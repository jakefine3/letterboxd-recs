# letterboxd movie recommendation system

a recommendation system built for letterboxd users.

## features

- csv import: load your letterboxd export data
- content-based filtering: recommendations based on movie attributes such as genre, cast, and director
- collaborative filtering: learn from similar users' (or friends') tastes
- hybrid approach: weighted combination that prioritized personal ratings
- coming soon: web scraper for multi-user data colection, web app acting as a public-facing service
